## 📌 DESCRIÇÃO DO PROJETO
Este projeto foi desenvolvido na linguagem C como parte da disciplina de Estrutura de Dados.

O objetivo do trabalho é aplicar conceitos fundamentais da disciplina, como organização e manipulação de dados, utilizando arquivos no formato CSV para armazenamento e leitura de informações.

Os programas implementam operações de entrada e saída de dados, permitindo persistência das informações mesmo após a execução, simulando o funcionamento de estruturas de dados aplicadas a um sistema real.
